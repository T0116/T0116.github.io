博客教程

如何用Git更新博客，超级详细的步骤

打开这个地址，安装hugo的地址

右击

进入如下界面

在美元符号后面输入：hugo new post/2020-8-12.md

2020-8-12是文件名

接着输入：

hugo –theme=hugo-lamp –baseUrl="https://T0116.github.io” –buildDrafts

=后面是主题名，””里面是github配置的仓库，–Drafts是草稿

cd public

接着输入：git add .

继续输入：

git commit -m “yyyy/mm/dd-hh:mm”

自动生成时间，防止提交重复，我这里是用的别的名称

最后，输入后需要等待一会儿，会弹出提示框输入账户密码：

git push -u origin master

输入成功的界面

大功告成！

删除博客就是把content/post/下的md文件删除和public/post/下的同名文件夹删除再次上传即可

期间出现了一个错误，我没联网，所以上传不成功，真是服了自己。
